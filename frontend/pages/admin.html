Execute the seat allocation algorithm (calls AllocateSeats() stored procedure).

⚙

Execute Allocation Round

This will run the AllocateSeats() MySQL stored procedure which assigns seats to students based on their JEE rank, category, and preference order. This action cannot be undone.

How allocation works:
  1. Students are sorted by JEE rank (lower = better).
  2. For each student, choices are evaluated in preference order.
  3. A seat in the student's category is reserved if available.
  4. Allocated students' choice status updates to Allocated.
  5. Remaining choices are marked Withdrawn.
